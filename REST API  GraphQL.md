|**API**|**Purpose**|
|---|---|
|**REST API**|Create, read, update, delete WP content via HTTP|
|**Settings API**|Build custom admin settings pages|
|**Options API**|Store/retrieve plugin/theme options|
|**Widgets API**|Create custom sidebar widgets|
|**Customizer API**|Add live-editing options to the WP Customizer|
|**Database API ($wpdb)**|Run custom SQL queries safely|
|**Shortcode API**|Add reusable code snippets via `[shortcode]`|
